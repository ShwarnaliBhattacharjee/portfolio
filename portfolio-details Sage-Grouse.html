Project information

  • Category: Spatial Analysis
  • Project date: Dec 2024 - March 2025
  • Data Source: Sagebrush Cover, Shurb Cover, NED DEM raster, NLCD Landcover
  • Tool Used: ArcGIS Pro (Spatial Analyst, Spatial Map Series), Python (Arcpy)

Excuted spatial analysis for Audubon Washington to advance Greater Sage-Grouse conservation by creating 394 data-driven 2x2 km survey blocks and 37 (5x5 mile) priority grids in Toppenish Ridge and Rattlesnake Hill Management Area of Washington State. Integrated sagebrush cover, NLCD land cover, NED DEM, and species sighting data using ArcGIS Pro and Python to identify priority survey blocks. Developed custom python scripts to automate habitat suitability analysis and randomly select 40 optimal transect cells per block. The project significantly enhanced field survey efficiency, improved spatial coverage of critical habitats, and directly informed conservation priorities and resource allocation.
